- Output은 사용자 소유
- 비공개 콘텐츠는 운영 목적 외 사용 없음

### 5️⃣ 보안 기능

|------|------------|------|

- 전송: TLS 1.2+
- 저장: AES-256

<details>
<summary>Capcut 보안검토</summary>

### 1️⃣ 공식 정보보안 인증
- **평가:** 미흡
- **내용:** SOC 2, ISO 27001 등 공식 보안 인증 획득 여부 확인 불가
- **리스크:** 보안 관리 체계 검증 불가

### 2️⃣ 데이터 저장 위치
- **평가:** 양호
- **내용:** 말레이시아, 싱가포르, 미국 등 해외 인프라 사용
- **이슈:** 개인정보 해외 이전 가능성 존재

### 3️⃣ 입력 데이터 학습 활용 여부
- **평가:** 미흡
- **내용:** 사용자 데이터의 AI 학습 활용 여부 확인 불가
- **리스크:** 기업 데이터 2차 활용 가능성 배제 불가

### 4️⃣ 이용약관 및 개인정보 처리 정책
- **평가:** 미흡
- **핵심 내용 요약:**
  - 사용자 콘텐츠가 기업 기밀로 취급되지 않음
  - 서비스 운영·개선 목적의 콘텐츠 활용 가능
  - 글로벌 서비스 제공을 위한 데이터 이전 가능
- **리스크:** 기업 콘텐츠 기밀성 보장 어려움

### 5️⃣ 보안 기능
- **평가:** 미흡
- 자체 MFA 기능 제공 없음  
- Google 로그인 사용 시에만 구글 계정 2단계 인증 간접 적용 가능  
- 기업용 보안 기능(SSO, 감사 로그 등) 미제공

### 6️⃣ 데이터 암호화
- **평가:** 미흡
- 전송 및 저장 데이터 암호화 방식 명시적 확인 불가

<a id="Cutback-보안검토"></a>
<details>
<summary>Cutback 보안검토</summary>

### 1️⃣ 공식 정보보안 인증
- **평가:** 양호
- **내용:** Creative Cloud, Document Cloud 등 엔터프라이즈 솔루션에 대해 SOC 2 Type II, ISO 27001, HIPAA Ready 등 다수 인증 보유

### 2️⃣ 데이터 저장 위치
- **평가:** 확인불가
- **내용:** 인프라 및 네트워크 보안 내용은 확인 가능하나, 특정 리전 선택 정책에 대한 명시적 언급은 부족함

### 3️⃣ 학습 활용 여부
- **평가:** 양호
- **내용:** Adobe는 사용자가 작성한 콘텐츠(로컬, 클라우드 포함)를 AI 학습용으로 사용하지 않음을 명시함

### 4️⃣ 이용약관 및 개인정보처리방침 상 데이터 처리 조항
- **평가:** 양호
- **내용:** Generative AI 기능 사용 시 입력 및 출력 데이터 모두 사용자 콘텐츠로 분류되며, ToS에 처리 방식이 명시됨

### 5️⃣ 보안 기능
- **평가:** 양호
- **내용:** Adobe Secure Product Lifecycle(SPLC) 적용, OWASP Top 10 기반 빌트인 보안 및 사고 대응 프로세스 운영

### 6️⃣ 데이터 암호화
- **평가:** 양호
- **내용:** PDF 문서 서명 시 PKI 기반 디지털 ID 지원 및 전송/저장 시 암호화 표준 적용

</details>
